Basic Sorting

Posted on 2018-05-02

这篇文章主要是回顾基础的排序算法

Bubble Sort 冒泡排序

通过相邻元素的比较和交换把小的数交换到最前面。
先确定最后一位(最大值)
Step:

  1. 比较相邻的元素。如果第一个比第二个大，就交换他们两个。
  2. 对每一对相邻元素作同样的工作，从开始第一对到结尾的最后一对。在这一点，最后的元素应该会是最大的数。
  3. 针对所有的元素重复以上的步骤，除了最后一个。
  4. 持续每次对越来越少的元素重复上面的步骤，直到没有任何一对数字需要比较。

Analyze:

  • Average Time complexity: O(n^2)
  • Space complexity: O(n)
  • Auxiliary Space: O(1)
  • Stable: Yes

Insertion Sort 插入排序

像扑克摸牌一样，把新的值插入有序的list中

Step:

  1. 从第一个元素开始，该元素可以认为已经被排序
  2. 取出下一个元素，在已经排序的元素序列中从后向前扫描
  3. 如果该元素（已排序）大于新元素，将该元素移到下一位置
  4. 重复步骤3，直到找到已排序的元素小于或者等于新元素的位置
  5. 将新元素插入到该位置后
  6. 重复步骤2~5

Analyze:

  • Average Time complexity: O(n^2)
  • Space complexity: O(n)
  • Auxiliary Space: O(1)
  • Stable: Yes
  • Can be improved by binary insertion sort

Selection Sort 选择排序

在无序的list中重复寻找最小值，并将其放到前面

Analyze:

  • Average Time complexity: O(n^2)
  • Space complexity: O(n)
  • Auxiliary Space: O(1)
  • Stable: Default implementation is not stable

Merge Sort 归并排序

分治法(Divide and Conquer)的典型例子, 主要实现合并两个有序数列

Analyze:

  • Average Time complexity: recursive algorithm and time complexity can be expressed as: $$T(n) = 2T(n/2) + \theta(n)$$
    Can be solved in O(nlogn)
  • Space complexity: O(n)
  • Auxiliary space: O(n), O(1) with linked lists
  • Stable: Yes

Quick Sort 快速排序

同样用分治法。

Steps:

  1. 选择一个基准值,可以选择list中第一位或最后一位作为基准,也可以随机
  2. 进行分区(partition), 把小于基准值的放到左边,大的放到右边
    1. LeftMark 从左往右寻找大于pivot的值, RightMark从右往左寻找pivot的值
    2. 如果LeftMark< RightMark,交换位置, 否则代表已经交换结束
    3. 最后将pivot交换到RightMark的位置
  3. 对两个分区进行递归

Analyze:

  • Average Time complexity: O(nlogn)
  • In-place Space complexity: O(logn)
  • Stable: In-place sort is not stable
