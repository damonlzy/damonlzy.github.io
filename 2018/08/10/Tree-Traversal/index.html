Posted on 2018-08-10

二叉树前中后序遍历

三种遍历都属于DFS

先序遍历顺序: root, left child, right child
Preorder: 1->2->4->5->3

中序遍历顺序: left child, root, right child
Inorder: 4->2->5->1->3

后序遍历顺序: left child, right child, root
Postorder: 4->5->2->3->1

递归实现逻辑很简单，非递归实现则需要用到Stack

BFS: 1->2->3->4->5
用Queue实现

Node class

先序遍历 Preorder

  • Recursion
  • Without Recursion
    1. 先把root 压入栈
    2. 当栈不为空时弹出并打印val
    3. 依次把右边和左边的child压入栈
    4. 重复2，3

中序遍历 Inorder

  • Recursion
  • Without Recursion
    1. current node = root
    2. current node压进栈，并让current 等于left child 直到current is Null，这样就到了leftmost child
    3. 如果current is Null 但是栈不为空
      1. 弹出最上面的Node
      2. 打印Node的value，current指向Node的right child
      3. 重复步骤3
    4. 如果current和栈都为空，则表示遍历结束

后序遍历 Postorder

  • Recursion

  • Without Recursion

    1. 用两个栈

      1. 将root 压进第一个栈
      2. 循环第一个栈直到为空
        1. 从第一个栈中弹出并压进第二个栈
        2. 分别将left child和right child压进第一个栈
      3. 弹出第二个栈的Node并打印

  1. 用一个栈

    1. 建一个栈
    2. 当root不为Null时
      1. 将right child 压进栈，再将root也压进栈
      2. root变为root的left child
    3. root等于栈中弹出最上面的Node
      1. 如果刚弹出的Node/root有right child 而且他的right child 是栈中最上面的，则弹出right child，再将root压入栈并设root等于right child
      2. 否则打印root的值并设root等于Null
    4. 重复2 3步骤，直到栈为空
